Danny Lau



Ever since college, I've customized my own computers and was interested in software because I wanted to know more about the computers I was building. Later when I started working I started getting into building websites and became interested in learning programming languages.
I use architectures such as Ruby on Rails and React.js to make web apps. My most recent project uses movement algorithms to randomize coordinates sets and routes, which in conjunction with the Google Maps API simulates dynamic movement on a map.

Fatebook.xyz

  • Robot-themed social tracking hub inspired by Facebook.com
  • • Web app built on Flux architecture and serving a RESTful JSON API
  • • Wrote algorithms to randomize coordinate sets and routes, in conjunction with the Google Maps API simulates dynamic user movement
  • • Single page structure utilizes React.js components to provide real-time search, tracking and untracking, and user comments
  • GitHub: https://github.com/catcher23/fatebook

JS Chess

  • • Move algorithms display all possible moves for a turn from a given position
